## What is Fungi Farmer?
Fungi Farmer is a dashboard prompt that was developed to display the predicted growth based on a constant cultivation condition and the usersâ€™ input on the length of the oyster mushroom bunch. The prompt uses the XGB algorithm for forecasting growth.
